# HotelReservationSystem
A simple Java-based Hotel Reservation System demonstrating core OOPs concepts (Encapsulation, Inheritance, Polymorphism, Abstraction) along with JDBC connectivity for managing hotel bookings. The project allows users to add customers, check room availability, make reservations, and view booking details while interacting with a MySQL database.
